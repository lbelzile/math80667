1 Introduction to texmex:

The goal of this tutorial is to show how one can estimate the probability of rare multivariate events with the conditional extremes methodology of (Heffernan:2004?) using the texmex package, whose name stands for “threshold exceedances and multivariate extremes”.

Download and install R (current version 4.1.1, nicknamed “Kick Things”) and an integrated development environment such as RStudio.

packages <- c("poorman", #a lightweight alternative to the tidyverse
              "lubridate", # easily converts dates
              "extRemes", # contains the Fort Collins data
              "patchwork", #combine ggplot objects
              "gridExtra", # data on grid
              "texmex") # threshold exceedances and multivariate extremes
# Install R packages from the list if not already on your machine
pkg_list <- packages[!packages %in% installed.packages()[,"Package"]]
for(pkg in seq_along(pkg_list)){
  install.packages(pkgs = pkg)
}
library(poorman)
library(texmex) # loads grammar of graphics
library(patchwork)
library(gridExtra) # diagnostic plots don't work without this

For illustration, we consider daily weather data in Fort Collins, Colorado from (Katz:2002?). The records are taken from the extRemes package and transformed to metric units. We focus on minimum and maximum daily temperature and associated precipitation. The data can be loaded by calling .

To simplify things, we will ignore serial dependence and extremal clustering and consider a conditional model given extreme precipitation (mostly as a way to discard the zero-inflated rainfall measurements from the analysis).

# load data
data(FortCollins, package = "texmexTutorial")
